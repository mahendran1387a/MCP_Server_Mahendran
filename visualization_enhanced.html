üêõ The Problem

Error: "Connection closed" when initializing MCP client

Root Cause:

  • Web server opens ChromaDB for RAG system
  • MCP server (subprocess) also tries to open ChromaDB
  • ChromaDB uses file locks - only ONE process allowed
  • Second connection fails ‚Üí MCP server crashes
  • User sees misleading "Ollama is not running" error

‚úÖ The Solution

Fix: MCP server uses HTTP API instead of direct database access
